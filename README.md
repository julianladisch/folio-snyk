A daily job uploads the front-end dependencies to Snyk: [.github/workflows/snyk.yml](.github/workflows/snyk.yml)

At Snyk they get monitored for security vulnerabilities.

The [FOLIO Security Team](https://folio-org.atlassian.net/wiki/spaces/SEC) monitors the results and maintains the [FOLIO Snyk account](https://folio-org.atlassian.net/wiki/spaces/SEC/pages/3443703/Snyk).
